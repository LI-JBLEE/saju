## DESIGN PHILOSOPHY

고급스럽고 차분한 동양적 감성을 현대적인 미니멀 디자인으로 표현한다.
복잡한 장식보다 여백과 타이포그래피로 품격을 만든다.
사용자는 리포트 내용에만 집중할 수 있어야 한다.

## PAGE STRUCTURE

### ① 메인 페이지 — `/`
중앙 정렬 단일 카드 레이아웃. 앱 타이틀 + 한 줄 설명 + 입력 폼.
배경: Deep Night. 카드: Midnight.
모바일 우선 설계, max-width: 480px.

### ② 로딩 — 리포트 생성 중
전체 화면 오버레이. 골드 색상 인디케이터 + "사주를 읽고 있습니다..." 텍스트.
팁 문구를 3초마다 순환 표시 (오행 설명 등 사주 상식).

### ③ 리포트 페이지 — `/report/[id]`
섹션별 카드 구성. 스트리밍으로 위에서 아래로 순차 출력.
상단 고정 헤더: 생년월일 요약 + 공유 버튼.
max-width: 720px, 중앙 정렬.

## RESPONSIVE DESIGN

- 모바일 (`~480px`): 단일 컬럼, 폰트 14px
- 태블릿 (`~768px`): 단일 컬럼, 폰트 15px
- 데스크탑 (`768px~`): max-width 720px 중앙 정렬
- 모든 터치 타겟 최소 `44px` 이상

## UI DO / DON'T

**해야 할 것**
- 여백을 충분히 사용
- 골드 컬러는 포인트로만
- 섹션 구분은 선이 아닌 여백으로
- 애니메이션은 subtle하게

**하지 말아야 할 것**
- 형광색 / 과도한 컬러 사용
- 팝업 / 모달 남발
- 자동 재생 콘텐츠
- 불필요한 네비게이션 바

## ANIMATION GUIDELINES

- 모든 트랜지션: `duration 200ms, ease-out`
- 리포트 스트리밍 텍스트: 자연스럽게 append (타이핑 효과 없음)
- 섹션 등장: `fade-in + translateY(8px → 0)`, 300ms
- 로딩 인디케이터: 골드 색상 pulse 애니메이션
- 과도한 애니메이션 금지 — 콘텐츠 집중 방해 요소 제거
