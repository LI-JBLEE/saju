# docs/product-specs/index.md
사주 리포트 웹앱 — 제품 스펙 정의

---

## PRODUCT SUMMARY

사용자가 생년월일시와 성별을 입력하면 사주를 계산하고, AI가 종합 분석 리포트를 생성하여 즉시 보여주는 웹앱.
로그인 없이 누구나 무료로 사용 가능하며, 결과는 고유 URL로 공유할 수 있다.

---

## USER JOURNEY

```
① 메인 페이지 접속
   └─ 생년월일 / 태어난 시간 / 성별 입력
② "사주 보기" 버튼 클릭
   └─ 로딩 화면 (리포트 생성 중...)
③ 리포트 페이지 이동 (/report/[id])
   └─ 섹션별 스트리밍 출력
④ 공유 버튼으로 URL 복사
```

---

## INPUT SPEC

- **생년월일**: 양력 기준, 연/월/일 선택 (1900년 이후)
- **태어난 시간**: 시간대 선택 (모름 선택 가능 → 시주 제외 분석)
- **성별**: 남 / 여

---

## REPORT STRUCTURE

### 섹션 1 — 사주 기본 정보
년주·월주·일주·시주 (천간·지지), 오행 구성 비율, 용신/기신

### 섹션 2 — 타고난 성격 & 기질
일간 기준 성격 분석, 강점과 약점, 대인관계 스타일

### 섹션 3 — 직업 & 재물운
적성에 맞는 직업군, 재물을 모으는 방식, 재물운 흐름

### 섹션 4 — 연애 & 결혼운
연애 스타일, 이상형, 결혼 시기 및 배우자 특징

### 섹션 5 — 건강운
오행 불균형에 따른 주의 부위, 건강 관리 조언

### 섹션 6 — 올해 운세 (당해 연도)
세운 분석, 상반기/하반기 흐름, 주요 키워드

### 섹션 7 — 종합 조언
현재 시점 기준 인생 전반 조언, 주의사항, 행운의 방향/색상/숫자

## SHARING FEATURE

- 리포트마다 고유 URL 생성: `/report/[uuid]`
- "링크 복사" 버튼으로 클립보드에 복사
- 링크 접속 시 해당 리포트 바로 조회 가능 (재생성 없음)

---

## OUT OF SCOPE (추후 고려)

- 로그인 / 회원가입
- 리포트 히스토리 관리
- 궁합 보기 (두 사람 비교)
- PDF 다운로드
- 유료 결제 기능

---

## SUCCESS METRICS

- 리포트 생성 완료율 95% 이상 (에러 없이 끝까지 생성)
- 리포트 생성 시간 30초 이내
- 공유 링크 정상 조회 100%
